General Overview

One Board Does It All

Discover the potential of our ESP32-C6 based development board, engineered to simplify your prototyping needs with integrated components that streamline connections and enhance functionality. Equipped with advanced features like WiFi 6, BLE 5.0, USB-C programming, and a robust power management system, this board is ready to support high-performance applications in robotics, IoT, and smart devices. Built-in components, including high-power motor drivers, addressable LEDs, and a Qwiic/Stemma QT port, minimize wiring needs and maximize reliability, allowing you to accelerate development with powerful, all-in-one hardware.

Designed by Rupak Poddar. For more information, visit the developer's portfolio at rupakpoddar.github.io.

Arduino Integration Guide

Step 1

This is a one-time setup. Open Arduino IDE and go to Settings/Preferences.

Step 2

In Additional Board Manager URLs, paste:

https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

Step 3

Go to Tools > Board > Board Manager.

Step 4

In the search box, type esp32 and install the package by Espressif Systems.

Step 5

Connect your board via USB-C, select ESP32C6 Dev Module, and choose the port.

High Power Motor Driver

Integrated motor drivers simplify robotics projects, allowing for dual motor control with high-power output pins directly from the board.

Motor Driver Setup Code


    // Define motor control pins
    #define MOTOR_1A 18 // Motor 1, Input A
    #define MOTOR_1B 19 // Motor 1, Input B
    #define MOTOR_2A 20 // Motor 2, Input A
    #define MOTOR_2B 21 // Motor 2, Input B

    void setup() {
        pinMode(MOTOR_1A, OUTPUT);
        pinMode(MOTOR_1B, OUTPUT);
        pinMode(MOTOR_2A, OUTPUT);
        pinMode(MOTOR_2B, OUTPUT);
    }

    void loop() {
        analogWrite(MOTOR_1A, 200); 
        analogWrite(MOTOR_1B, 0); 
        delay(2000);
        analogWrite(MOTOR_1A, 0);
        analogWrite(MOTOR_1B, 0);
        delay(1000);
        analogWrite(MOTOR_2A, 0);
        analogWrite(MOTOR_2B, 150);
        delay(2000);
        analogWrite(MOTOR_2A, 0);
        analogWrite(MOTOR_2B, 0);
        delay(1000);
    }

Qwiic/Stemma Support

The onboard Qwiic connector supports I2C modules from SparkFun and Adafruit, making it easy to expand with additional sensors and peripherals.

Qwiic Setup Code


    #include <Wire.h>
    #include "libraryName.h"

    void setup() {
        Wire.begin(12, 13);
        libraryName.begin(Wire);
    }

Addressable LEDs

Built-in WS2812B addressable LEDs offer customizable lighting options, perfect for adding visual feedback or aesthetic appeal to your projects.

WS2812B NeoPixel Setup Code


    #include <Adafruit_NeoPixel.h>

    #define NUM_LEDS 4
    #define DATA_PIN 4

    Adafruit_NeoPixel leds(NUM_LEDS, DATA_PIN, NEO_GRB + NEO_KHZ800);

    void setup() {
        leds.begin();
        //
        // ...
        //
        leds.show();
    }

FastLED Setup Code


    #include <FastLED.h>

    #define NUM_LEDS 4
    #define DATA_PIN 4

    CRGB leds[NUM_LEDS];

    void setup() {
        FastLED.addLeds<NEOPIXEL, DATA_PIN>(leds, NUM_LEDS);
        //
        // ...
        //
        FastLED.show();
    }
