Sync Link Privacy Policy

Last Updated: 15 November 2023

1. Information We Collect:

No User Data Collected: Sync Link does not collect any user data. The app only requires users to select a BLE device, provide UUIDs, or enter a URL for communication.

2. Data Usage:

Device Interaction: Sync Link uses the provided BLE device information or URL exclusively for the purpose of device interaction. No user data is collected, stored, or shared.

3. Data Security:

Security Measures: Sync Link does not explicitly implement additional security measures. The app makes GET requests to the specified URL or sends a string over BLE. Users are advised to ensure the security of their communication channels.

4. Changes to This Privacy Policy:

Updates: We may update our Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this policy. Please review it periodically.
